Telemetry Sanitizer Project

A simple exercise in string sanitization while learning C#.
Dependencies

This project requires the Humanizer dependency.
What is this project?

I created this project as an exercise to try and learn some new concepts like methods while I was learning C# and the .NET ecosystem from Microsoft Learn courses. As you can see, at my current point in the course I still have not learned about lists, arrays, etc. so most of it is very primitive and hardcoded. At this point, I have no idea about methods either, so I made do with stack overflow pages about how to reference variables between methods so that I could learn how this all works. Is the code perfect? No. I am still very much early into learning what I need to know about Azure DevOps and C#, so this was an exercise to try and squeeze everything I knew to its last drops and incorporate something I hadn't known before.

The result is the code you see before you. Not perfect, but just a trial. I learned a lot about debugging and compiling the code from this exercise, and as you can see this is the second ever published repo to my GitHub account, so this is also an exercise in git and source control procedure.

Nevertheless, I hope that this repository and future ones will show you my learning process, and I hope that you will follow along as I wrap my head around Software Engineering.
